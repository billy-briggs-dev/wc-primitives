WC Primitives Demo - Alert Dialog

Headless Web Component Primitives built with Lit 3

Destructive Action Confirmation

Alert dialogs require explicit user action and cannot be dismissed by clicking outside or pressing Escape.

Delete Account Are you absolutely sure? This action cannot be undone. This will permanently delete your account and remove your data from our servers.
Cancel Yes, delete account
Try this: Open the dialog and try clicking outside or pressing Escape. Unlike regular dialogs, alert dialogs require you to explicitly click Cancel or the action button.

File Deletion

A simpler confirmation for file deletion.

Delete File Delete "document.pdf"? This file will be permanently deleted. You won't be able to recover it.
Cancel Delete File

Event Tracking

Alert dialogs dispatch events that you can listen to.

Key Difference from Dialog: Alert dialogs use role="alertdialog" and cannot be dismissed by clicking outside or pressing Escape. This ensures important confirmations are not accidentally dismissed. Users must explicitly choose an action.
Note: All styling you see is applied by the demo page. The components themselves are headless and come with no default styles. You have complete control over the appearance!
