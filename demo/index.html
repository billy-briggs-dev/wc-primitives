WC Primitives Demo

Headless Web Component Primitives built with Lit 3

Single Mode Accordion (Collapsible)

Only one item can be open at a time, and clicking an open item closes it.

What is WC Primitives? WC Primitives is a library of headless web components built with Lit 3. These components are framework-agnostic and provide accessible, composable UI primitives. Why headless components? Headless components provide behavior and accessibility without imposing styles. This gives you complete control over the appearance while ensuring proper functionality and accessibility out of the box. How do I use them? Simply import the components in your HTML or JavaScript, then style them with CSS. They work in any framework (React, Vue, Angular, etc.) or vanilla JavaScript.

Multiple Mode Accordion

Multiple items can be open at the same time.

Accessible by Default All components follow WAI-ARIA patterns and include proper keyboard navigation, focus management, and screen reader support. Framework Agnostic Built on Web Standards, these components work everywhere - React, Vue, Angular, Svelte, or vanilla JavaScript. No framework lock-in. Fully Composable Components are designed to be composed together, giving you flexibility to create complex UI patterns while maintaining consistency.
